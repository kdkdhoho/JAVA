# 1. 연산자(operator)

## 1.1 연산자와 피연산자

## 1.2 식(式)과 대입연산자

연산자와 피연산자를 조합하여 계산하고자하는 바를 표현하는 것을 **식(expression)**이라고 한다.

식을 계산하여 결과를 얻는 것을 **식을 평가(evaluation)**한다고 한다.

하나의 식을 평가(계산)하면, 단 하나의 결과를 얻는다.
